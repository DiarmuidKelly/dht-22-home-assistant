# 3D Printed Enclosure

This directory contains STL files for a 3D printed enclosure for the Raspberry Pi Pico W with DHT22 sensor.

## Files

- `Pico_TempHum_BTM.stl` - Bottom part of enclosure
- `Pico_TempHum_TP.stl` - Top part of enclosure

## Usage

1. Download the STL files
2. Slice with your preferred slicer (Cura, PrusaSlicer, etc.)
3. Print with PLA or PETG
4. Assemble the enclosure around your Pico W + DHT22 sensor

## Notes

- Ensure adequate ventilation for accurate temperature readings
- The DHT22 sensor should be positioned for proper airflow
- Modify the design if needed (remix allowed under CC BY-NC-SA 4.0)
